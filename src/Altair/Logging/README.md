# Univeros Logging

> PSR-3 application logging for Altair, backed by Monolog and wired from `LOG_*` env vars

## Installation

```bash
composer require univeros/logging
```

## Documentation

The full guide for this package lives in the [Univeros documentation](https://github.com/univeros/docs/blob/master/packages/logging.md).
